# LLM 教學

本專案是 LLM 的教學課程，包含以下內容：

## 課程內容

- **01-Gemini 介紹.ipynb** - Google Gemini AI 模型的介紹與使用
- **02-LangChain 入門.ipynb** - LangChain 框架的基礎教學

## 環境設定

1. 複製環境設定檔案：
   ```bash
   cp .env.sample .env
   ```

2. 在 `.env` 文件中設定你的 API 金鑰

## 使用方式

使用 Jupyter Notebook 開啟相應的 `.ipynb` 文件進行學習。
